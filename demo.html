No_Class Demo

No_Class is a modern stylesheet driven exclusively by native markup. Does you project have No_Class ? Get started by visiting the Github page and reviewing this demo page.



To get started, download the latest from Github or paste the following in the <head> tag of your HTML or PHP file:

                <link rel="stylesheet" href="https://raw.githubusercontent.com/madebysnacks/No_Class/main/no_class.css">
